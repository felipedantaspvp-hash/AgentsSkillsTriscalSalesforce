# Triscal Agents & Skill Updater

Extensão VSCode para atualizar automaticamente os arquivos padrão Triscal no projeto aberto.

## Comando

- `Triscal: Atualizar Agents e Skill Triscal`

## O que o comando faz

Copia estes arquivos para o workspace aberto:

- `AGENTS.md`
- `AI_HANDLERS_TRISCAL.md`
- `agents/skills/triscal-salesforce/SKILL.md`

## Desenvolvimento rápido

1. Abra a pasta `vscode-extension` no VSCode.
2. Pressione `F5` para abrir a Extension Development Host.
3. Em outro workspace aberto, execute o comando `Triscal: Atualizar Agents e Skill Triscal`.
